## **Backend Setup (Node.js + Express)**

### Go to the backend folder:

```bash
cd backend
```

### Install dependencies:

```bash
npm install
```

### Start the server:

```bash
node server.js
```

> Server will run at: `http://localhost:5000`

---

## **Frontend Setup (React)**

### Go to the frontend folder:

```bash
cd frontend
```

### Install dependencies:

```bash
npm install
```

### Start the frontend:

```bash
npm start
```

> React app will open at: `http://localhost:3000`
